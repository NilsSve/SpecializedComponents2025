XML Sample-5

 

This samples shows the basics of XSL transformation. The sample does the following:

 

  1. Creates an XML Object
  2. Assigns the document to an existing XML document and loads it
  3. Creates an XSL object (which is another XML document)
  4. Applies an XSL transformation to the XML document, returning a string representing HTML markup.
  5. Save the HTML document and display it in the browser.

 

Performing an XSL transformation is simple.

 

You must first create and load an XML document and find the node in this document where you want to transformation to begin (XSL transformation is applied to the XML node and all of its child nodes). Often that starting node will be your root (or document element) node in which case the entire XML document is transformed.  

 

Next you must define and load the XSL node that is used for the transformation. In this sample we will create a second XML document which is this XSL node. Note that XSL is defined using XML, so an XSL document can always be loaded using the XML parser.

 

Finally transform the data use the message Get XSLTransformation

 

Get XSLTransformation of hoRoot hoXSL to sData

 

Where: hoRoot is the XML document node to start the transformation at

            hoXSL is the node of the of XSL document

            sData  is the resulting, transformed string.

 

In this sample, the XSL document will transform the XML document into HTML. The XSL document looks something like this:

 

<?xml version='1.0'?>

<xsl:stylesheet version="1.0" xmlns:xsl="http://www.w3.org/1999/XSL/Transform"

   xmlns:custList="http://www.dataaccess.com/CustomerList">

<xsl:template match="custList:CUSTOMERLIST">

    <HTML>

<BODY>

<TABLE BORDER = "1">

    <TR>

<TD>NAME</TD>

<TD>SALES REP</TD>

<TD>TELEPHONE</TD>

    </TR>

    <xsl:for-each select="custList:CUSTOMER">

    <TR>

<TD><xsl:value-of select="@NAME"/></TD>

<TD><xsl:value-of select="custList:SALES-REP"/></TD>

<TD><xsl:value-of select="custList:TELEPHONE"/></TD>

    </TR>

    </xsl:for-each>

</TABLE>

</BODY>

    </HTML>

</xsl:template>

</xsl:stylesheet>

 

XSL is a complete template transformation language and is a integral part of the W3c XML specifications.
